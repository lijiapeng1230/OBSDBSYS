概述

本文档描述了基于破天星辰科技网络接口盒子时间接口的时钟控件，适用于OBS等直播软件的右上角时间显示。

控件特性

• 实时显示当前时间（时:分:秒）

• 显示当前日期（年-月-日）

• 支持网络时间同步（接口盒子API）

• 网络不可用时自动回退到本地时间

• 半透明黑色背景设计，适合各种直播场景

配置参数

API配置

参数名 类型 默认值 说明

apiUrl string https://cn.apihz.cn/api/time/getapi.php 时间API接口地址

apiParams.id string 88888888 API接口ID

apiParams.key string 88888888 API接口密钥

apiParams.type string 2 API请求类型

样式配置

可通过修改CSS代码调整以下样式：

样式类 说明 可调整属性

.time-container 时间显示容器 background, border-radius, padding, width

.date-container 日期显示容器 background, border-radius, padding, width

.time-display 时间文字样式 font-size, color, letter-spacing

.date-display 日期文字样式 font-size, color

.status 状态提示文字样式 font-size, color

使用说明

1. 将HTML文件添加到OBS浏览器源中
2. 设置宽度为220px，高度为140px
3. 控件会自动尝试连接时间API服务器
4. 同步成功后显示"接口盒子api时间 (已同步)"
5. 若网络连接失败，自动切换至本地时间并显示"当前使用本地时间"

代码说明

HTML结构

<div class="time-container">
    <div class="time-display" id="timeDisplay">00:00:00</div>

<div class="date-container">
    <div class="date-display" id="dateDisplay">0000-00-00</div>

JavaScript功能

• fetchAliyunTime(): 从API获取网络时间

• fallbackToLocalTime(): 回退到本地时间

• updateTime(): 更新界面时间显示

自定义修改指南

修改尺寸

调整body元素的宽度和高度，以及相关容器的位置：
body {
    width: 300px;  /* 新宽度 */
    height: 160px; /* 新高度 */
}

.time-container {
    top: 15px;     /* 垂直位置调整 */
    left: 15px;    /* 水平位置调整 */
    width: 270px;  /* 新宽度 */
}


修改颜色主题

更改颜色变量可创建不同主题：
.time-display {
    color: #ff9900; /* 更改时间颜色 */
}

.date-display {
    color: #aaaaaa; /* 更改日期颜色 */
}


更换时间API

修改API配置部分可使用其他时间服务：
const apiUrl = 'https://your-time-api.com/endpoint';
const apiParams = {
    id: 'your_id',
    key: 'your_key',
    type: '2'
};


注意事项

1. 确保网络连接正常以获得准确的时间同步
2. 如需修改API参数，请确保使用有效的ID和密钥
3. 修改样式时注意保持背景半透明以确保在各种场景下都能清晰显示
4. 该控件设计尺寸为220×140像素，修改尺寸时需相应调整各元素位置

故障排除

• 若长时间显示"正在同步时间..."，请检查网络连接

• 若时间显示不正常，请检查API参数是否正确

• 若布局错乱，请检查CSS样式是否正确加载
